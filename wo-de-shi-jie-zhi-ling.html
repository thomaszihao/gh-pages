    死亡不掉落指令：/gamerule keepInventory true 命令方块指令：/give @s command_block

    给予一个人五块饼干:给予一个随机玩家一个物品并告诉所有人

    这个装置会给予一个随机玩家五个曲奇，然后告诉所有人。

    首先创建一个记分板(名字任意，注意除非在你测试，否则不要显示它)，比如说“cookie”

    /scoreboard objectives add cookie dummy cookie

    第一个命令方块的命令：

    /scoreboard players set @r cookie 1

    第二个和第三个命令（两者可颠倒，但要同时被激活）

    /say @a[scores={cookie=1}] 已被给予五个曲奇。

    /give @a[scores={cookie=1}] minecraft:cookie 5

    第四个命令方块的命令

    /scoreboard players set @a[scores={cookie=1}] cookie 0

    上图中I代表输入红石信号原。（建议可使用永动红石电路）

    允许方块：/give @s allow 拒绝方块：/give @s deny 屏障：/give @s barrier 边界方块：/give @s border_block。 我的世界无限刷方块的指令是“/give @a minecraft:command_block”。具体方法如下：

    1.首先我们需要按下“T”键，打开聊天栏，然后在聊天栏中输入指令：/give @a minecraft:command_block执行， 这样我们就能获得一个命令方块了，获得之后我们就可以进行下一步操作。

    2.然后找到我们想循环刷新方块的位置，这个时候摁住“F3”键然后记下图中所示位置的坐标，可以将这个时候的坐标打在聊天栏中方便查找。（如果你是想放置在脚下记得将y轴的坐标向下减一位）

    3.记住坐标之后我们需要把，命令方块中两个设置更改一下，如图所示，改为“循环”和“保持开启”。

    4.然后我们将指令 /setblock x y z block输入到命令方块之中，最后点击完成即可完成命令方块指令的设置。

    5.来到我们刚刚记录的坐标位置，无论怎么破坏这个方块，他都会自动循环刷新方块，这样我们就成功了。

/give @p minecraft:diamond_sword 1 0 {Unbreakable:1b,ench:[{id:16,lvl:10000}]}这条命令的意思是给你一个锋利10000且无法破坏的钻石剑，如果想加入更多的nbt标签建议使用命令方块，因为聊天栏是有字数限制的。以上指令仅限Java版（即电脑版）

    最后，怎么做出成品呢，就是把全部的都联合在一起

    /give 770019293 minecraft:blaze_rod 1 0 {ench:[{id:16,lvl:32767},{id:17,lvl:32767},{id:22,lvl:32767},{id:18,lvl:32767}],Unbreakable:1b,display:{Name:”如意金箍棒”,Lore:[“上古神铁所制 重十万八千斤 一棍下去 天崩地裂！”]},AttributeModifiers:[{Operation:0,UUIDLeast:1,UUIDMost:1,Amount:9000000000.0,AttributeName:generic.attackDamage,Name:Attack}]}

    中间要用逗号隔开，别忘记加括号，就可以了。

    《 我的世界 》这款游戏中的附魔9999指令是/give 你的名字 276 1 0 {ench: [ {id:输入代码,lvl:9999}]}，后面的9999就是你要附魔的数值了，随便改，一般是不会超出限制的，不一定要得是9999。

    我的世界武器附魔指令：

    在游戏中，小伙伴们自定义附魔用的NBT是{ench}，如果在指令方块里用这个NBT，就是让指令方块把刷出的物品进行自定义附魔。解释：

    ench：自定义附魔，要自定义附魔一定要打这个

    id：附魔属性ID。要附魔什么就打属性的ID

    lvl：附魔等级。如果不打等级只打ID，将会刷出一把0级附魔的剑，没任何意义。10级的话就是：lvl:10

    出了武器可以附魔，别的装备其实也可以附魔

    想要多个附魔属性，就在后面加个逗号就OK了

    举个例子：

    /give @p 276 1 0

    {ench:[{id:16,lvl:10},{id:17,lvl:10},{2:18,lvl:10},{id:19,lvl:10},{id:20,lvl:10},{id:21,lvl:10}]}

    大家用这个指令，就可以做出来一个6属性的10级钻石剑(≧▽≦)/啦

    然后如果还不满意，还想要更高级的，可以改下lvl等级(PS:最高级别24288级，不过，如果你把抢夺也设置成24288级，那等你显卡爆掉吧哈哈哈哈所以不可以贪心哟。
