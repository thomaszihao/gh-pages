我的世界指令

唐无双 16:50:24 2022年05月09日 指令大全

死亡不掉落指令：/gamerule keepInventory true 命令方块指令：/give @s command_block

给予一个人五块饼干:给予一个随机玩家一个物品并告诉所有人

这个装置会给予一个随机玩家五个曲奇，然后告诉所有人。

首先创建一个记分板(名字任意，注意除非在你测试，否则不要显示它)，比如说“cookie”

/scoreboard objectives add cookie dummy cookie

第一个命令方块的命令：

/scoreboard players set @r cookie 1

第二个和第三个命令（两者可颠倒，但要同时被激活）

/say @a[scores={cookie=1}] 已被给予五个曲奇。

/give @a[scores={cookie=1}] minecraft:cookie 5

第四个命令方块的命令

/scoreboard players set @a[scores={cookie=1}] cookie 0

上图中I代表输入红石信号原。（建议可使用永动红石电路）

允许方块：/give @s allow 拒绝方块：/give @s deny 屏障：/give @s barrier 边界方块：/give @s border_block。 我的世界无限刷方块的指令是“/give @a minecraft:command_block”。具体方法如下：

1.首先我们需要按下“T”键，打开聊天栏，然后在聊天栏中输入指令：/give @a minecraft:command_block执行， 这样我们就能获得一个命令方块了，获得之后我们就可以进行下一步操作。

2.然后找到我们想循环刷新方块的位置，这个时候摁住“F3”键然后记下图中所示位置的坐标，可以将这个时候的坐标打在聊天栏中方便查找。（如果你是想放置在脚下记得将y轴的坐标向下减一位）

3.记住坐标之后我们需要把，命令方块中两个设置更改一下，如图所示，改为“循环”和“保持开启”。

4.然后我们将指令 /setblock x y z block输入到命令方块之中，最后点击完成即可完成命令方块指令的设置。

5.来到我们刚刚记录的坐标位置，无论怎么破坏这个方块，他都会自动循环刷新方块，这样我们就成功了。

　/spawnpoint 设置重生点。 /scoreboard 计分板。 /experience 刷经验。 /team 修改队伍。 /give给予玩家物品。 /op [玩家名称] 给某个玩家OP

/deop [玩家名称] 取消某个玩家的OP /gm [0/1/2] [玩家名称] 给别人生存模式/创造模式/冒险模式 /setworth [物品名称/物品ID] [数量] 设置一个东西的价格

/worth [玩家名称] 查看价格，不写物品ID默认查看手上的物品价格

/itemdb 查询手上所拿物品的名称和ID

/mail [read/clear/send] [玩家名称] [语句] 读取/清除/发送某个离线玩家的留言 tp [玩家名称] 强制飞到某个玩家的身边

/tphere [玩家名称] 强制某个玩家飞到你身边

/tpall [玩家名称] 强制所有玩家飞到你身边 /baltop 查看财富排行榜 /fly [玩家名称] 给某个玩家飞行的权限

/invsee 查看并可以操作某个玩家的背包

/clearinventory [玩家名称] 清空某个玩家的背包 /vanish 隐身 /butcher（杀死附近所有的怪物） /gamemode 0 生存

　　/gamemode 1 创造

　　/gamemode 2 冒险

　　/gamemode 3 旁观 (旧版本不能使用)
